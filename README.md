# 🌑 MAKIMA M-DX v1.0.0🌑

Un bot WhatsApp **riche en fonctionnalités* développé avec **Node.js** et [Baileys](https://github.com/WhiskeySockets/Baileys).  
Il intègre l’automatisation, la gestion des médias, des outils de recherche, téléchargement et des commandes avancées & Automatique pour groupes.

> *𝚃𝙷𝙴 𝙳𝙰𝚁𝙺𝙽𝙴𝚂𝚂* – L’ombre veille sur toi.

## ⚡ 𝙻𝙰𝙽𝙲𝙴𝙼𝙴𝙽𝚃 𝙴𝚃 𝙸𝙽𝚂𝚃𝙰𝙻𝙻𝙰𝚃𝙸𝙾𝙽𝚂 𝙳𝚄 𝙱𝙾𝚃

1. **Cloner le dépôt :**
   ```bash
   git clone https://github.com/Danscot/makima-mdx.git
   cd makima-mdx
```

1. Installer les dépendances :
   ```bash
   npm install
   ```
2. Lancer le bot :
   ```bash
   npm start
   ```
